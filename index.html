My fav Game Type

Action-RPG

คือการผสมเกมแนว RPG กับ action เข้าด้วยกัน Role-Playing Game หรือ RPG คือแนวเกมที่เราจะได้สวมบทบาทเป็นตัวละครตัวหนึ่ง ทำหน้าที่ควบคุมทุกอย่างเกี่ยวกับตัวละครตามระบบเกม

First Person Shooting

เกมยิงที่เป็นมุมกล้องเสมือนเราสวมบทตัวละครนั้น มุมมองจากตัวเราเอง ผู้เล่นสวมบทบาทผ่านมุม​มองจากสายตาตัวละครตัวหนึ่ง

เกมกีฬา (Sport Game)

ตรงตัวเลยแหละ อย่างเกมที่เป็นเกมเกี่ยวกับกีฬา

MOBA

เกมต่อสุ้กันระหว่าง 2 ฝ่าย แบ่งทีม และเล่นกันหลายๆคนในตัวเกม เกมวางแผ่นแบบเรียลไทม์ให้ผู้เล่นแต่ละคน ควบคุมตัวละคร ซึ่งส่วนใหญ่เรียกว่า Hero เพื่อทำการต่อสู้กับอีกฝ่ายซึ่งจะเป็นผู้เล่นคนอื่นบังคับด้วยเช่นกัน

เหตุผลที่ชอบเล่นเกม

โดยส่วนตัวเป็นคนที่ชอบหาอะไรทำที่สนุกๆทั้งเล่นเกม เล่นดนตรี เล่นกีฬา อ่านการ์ตูน ส่วนเวลาเล่นเกมนั้นก็มีทั้งเล่นคนเดียวและเล่นกับเพื่อน บางทีกลับมาจากเรียนก็อยากที่จะเล่นเกมเพื่อผ่อนคลาย ช่วยทำให้สมองโล่งแถมยังได้ทำอะไรสนุกๆกับเพื่อนได้ง่ายๆอีกด้วย
